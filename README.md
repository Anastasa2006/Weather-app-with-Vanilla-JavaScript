# Weather-App-with-Vanilla-JavaScript
Веб-приложение для просмотра текущей погоды и прогноза на 7 дней в любом городе мира. Пользователь может ввести название города в поисковую строку, после чего приложение отправляет запрос к OpenWeatherMap API и отображает актуальные данные: температуру, влажность, скорость ветра, видимость и иконку погоды. Для городов с одинаковыми названиями реализован механизм выбора через модальное окно. Приложение сохраняет последний запрошенный город в localStorage.

## Основные функции
1. Поиск погоды в городе
* Ввод названия города в поисковую строку;
* Отображение текущей погоды: температура, влажность, ветер, видимость;
* Отображение прогноза на 7 дней с минимальной и максимальной температурой.


2. Умный поиск с поддержкой дубликатов
* Поиск городов через Geocoding API;
* При дубликатах названий открывается модальное окно с выбором города;
* Отображение кода страны для идентификации города.


3. Обработка неофициальных названий
* Автоматическое перенаправление неофициальных названий (Питер → Санкт-Петербург);
* Поддержка популярных сокращений (СПБ, МСК, ЕКБ и др.).


4. Динамический прогноз на неделю
* Получение данных с шагом 3 часа;
* Группировка по дням;
* Расчет минимальной и максимальной температуры для каждого дня;
* Отображение иконки погоды для каждого дня.


5. Сохранение состояния;
* Автоматическое сохранение последнего города в localStorage;
* Восстановление города при перезагрузке страницы.


6. Адаптивный дизайн
* Корректное отображение на всех устройствах;
* Темная тема с эффектами стекла;
* Оптимизация для мобильных устройств.

## Установка и запуск
### Локальный запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/ваш-username/weather-app.git
cd weather-app
```

2. Откройте файл index.html в браузере:
* Способ 1: Дважды кликните на файл
* Способ 2: Используйте Live Server в VS Code

4. Вставьте ваш API ключ в файл script.js :
```
const API_KEY = 'ваш_ключ_сюда';
```
### Получение API ключа OpenWeatherMap

1. Зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/)
2. Подтвердите email
3. Перейдите в раздел API Keys: [https://home.openweathermap.org/api_keys](https://home.openweathermap.org/api_keys)
4. Создайте новый ключ и вставьте его в script.js

## Архитектура
### Взаимодействие с API
```
Пользователь → Ввод города → Geocoding API → Получение координат
                                           ↓
Пользователь ← Отображение ← Weather API ← Получение погоды
                                           ↓
Пользователь ← Отображение ← Forecast API ← Получение прогноза
```

### Используемые API Endpoints
| API               | URL                      | Назначение                                   |
|-------------------|--------------------------|----------------------------------------------|
| Geocoding API     | /geo/1.0/direct          | Поиск города по названию                     |
| Current Weather   | /data/2.5/weather        | Текущая погода                               |
| 5 Day Forecast    | /data/2.5/forecast       | Прогноз на 5 дней (группируется в 7 дней)   |

## Особенности реализации
- **Обработка городов-дубликатов**: при поиске города, название которого существует в нескольких странах, открывается модальное
окно со списком всех вариантов. Пользователь может выбрать нужный город, после чего
загружается его погода;
- **Фильтрация результатов**: реализован черный список для исключения нерелевантных результатов поиска;
- **Кэширование**: результаты поиска городов кэшируются для оптимизации производительности и уменьшения
количества запросов к API;
- **Принудительное перенаправление**: для популярных неофициальных названий (Питер, СПб, МСК и др.) реализовано принудительное
перенаправление на официальное название с правильными координатами.
